# Identificatie
Conceptversie
{: .label .label-yellow }

Ieder begrip krijgt een betekenisloze, stabiele UUID (versie 4) toegekend. Deze wordt als identificerend kenmerk vastgelegd met [`dct:identifier`](http://purl.org/dc/terms/identifier).
